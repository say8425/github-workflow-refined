# Privacy Policy

**Last Updated: February 2025**

## Overview

GitHub Workflow Refined is a browser extension that enhances the GitHub Actions workflow pages. This privacy policy explains how we handle your data.

## Data Collection

### What We Collect

This extension collects **only user preference settings**, which include:

- Time display mode (relative, absolute, or auto)
- Time format pattern (e.g., "YYYY-MM-DD HH:mm:ss")
- Auto threshold for time display
- Language preference
- Today indicator toggle
- Auto-expand workflows toggle

### What We Do NOT Collect

This extension does **not** collect:

- Personal information (name, email, address, phone number)
- Authentication credentials or passwords
- Financial or payment information
- Health information
- Web browsing history or activity
- Website content or form data
- Cookies or tracking identifiers
- IP addresses or location data
- Any automatically collected data (no server logs, no HTTP logs, no analytics)

## Data Usage

The collected preference settings are used **solely** to:

- Remember your preferred time display format across browser sessions
- Apply your language preference to the extension interface
- Maintain your workflow display settings (auto-expand toggle)

The data is **never** used for:

- Advertising or marketing
- User profiling or tracking
- Sale to third parties
- Any purpose other than providing the extension's core functionality

## Data Storage

All settings are stored **locally** in your browser using `chrome.storage.local`. This data:

- Never leaves your device
- Is not transmitted to any external servers
- Is not encrypted (as it contains no sensitive information)
- Remains entirely under your control

## Data Sharing

This extension does **not** share any data with:

- The extension developer
- Third-party services
- Advertisers
- Analytics providers
- Any other external parties

## Data Retention

Your settings persist until:

- You manually reset them through the extension popup
- You uninstall the extension (data is automatically deleted by Chrome)
- You clear your browser's extension data

## Permissions

This extension requests only the minimum necessary permissions:

| Permission | Purpose | Scope |
|------------|---------|-------|
| `storage` | Save your preferences locally | Local browser only |
| Host permission | Modify GitHub Actions page display | `github.com/*/actions*` only |

We do **not** request permissions for:

- Reading browsing history
- Accessing other websites
- Reading or modifying bookmarks
- Accessing clipboard
- Capturing screen or audio

## Third-Party Services

This extension:

- Does not use any analytics services (no Google Analytics, no tracking pixels)
- Does not connect to any external APIs or servers
- Does not include any advertising SDKs
- Does not embed any third-party scripts

## Children's Privacy

This extension does not knowingly collect any information from children under 13 years of age.

## Changes to This Policy

Any updates to this privacy policy will be reflected in this document with an updated "Last Updated" date. Significant changes will be noted in the extension's changelog.

## Contact
